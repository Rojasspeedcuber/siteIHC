# Portfólio Profissional - Luiz Eduardo Crisanto

## 📋 Descrição do Projeto

Portfólio profissional responsivo e acessível de **Luiz Eduardo Crisanto**, desenvolvedor backend especializado em Java e Spring Boot. O site apresenta de forma profissional a carreira, experiências, projetos e hobbies do cliente.

## 🎯 Objetivo

Criar um site com 5 abas principais que represente bem o perfil profissional de Luiz Eduardo, seguindo padrões de usabilidade, acessibilidade e as heurísticas de Jakob Nielsen.

## 📑 Estrutura das Abas

1. **Home** - Apresentação inicial com foto profissional, headline e estatísticas rápidas
2. **Currículo (CV)** - Dados estruturados do currículo em formato visual e profissional
3. **Experiência** - Detalhamento da experiência profissional na Assembleia Legislativa de Pernambuco
4. **Projetos** - Portfólio de projetos técnicos e áreas de atuação
5. **Futebol** - Seção pessoal com imagens e informações sobre o hobby de futebol

## ♿ Acessibilidade

O site implementa as melhores práticas de acessibilidade:

- ✅ Navegação por teclado (Tab, Arrow Keys, Home, End, Escape)
- ✅ ARIA labels e roles semânticos
- ✅ Contraste WCAG AA mínimo (4.5:1)
- ✅ Alt text descritivo em imagens
- ✅ Suporte a leitores de tela
- ✅ Suporte a modo escuro (prefers-color-scheme)
- ✅ Respeito a preferência de movimento reduzido (prefers-reduced-motion)
- ✅ Skip link para conteúdo principal
- ✅ Focus visível em elementos interativos

## 📱 Responsividade

O site é totalmente responsivo e otimizado para:

- **Desktop** (1024px+) - Layout completo com navbar horizontal
- **Tablet** (768px-1023px) - Layout adaptado com navbar colapsável
- **Mobile** (< 768px) - Menu hamburger, stack vertical, touch-friendly
- **Small Mobile** (< 480px) - Otimizações adicionais para telas pequenas

## 🎯 Heurísticas de Nielsen Aplicadas

1. **Visibilidade do Status do Sistema** - Indicador visual da aba ativa com ponto amarelo
2. **Correspondência entre Sistema e Mundo Real** - Linguagem clara e profissional
3. **Controle e Liberdade do Usuário** - Navegação intuitiva, menu responsivo
4. **Prevenção de Erros** - Validação de entrada, feedback visual
5. **Estética e Design Minimalista** - Interface limpa, sem poluição visual
6. **Ajuda e Documentação** - Descrições claras, tooltips quando necessário
7. **Flexibilidade e Eficiência** - Atalhos de teclado, navegação rápida
8. **Reconhecimento vs Memorização** - Ícones e labels claros, navegação óbvia

## 🛠️ Como Usar

### Abrir o Site
1. Abra o arquivo `index.html` em um navegador web
2. Ou hospede em um servidor web (recomendado para HTTPS)

### Navegação
- Clique nos links da navbar para navegar entre abas
- Use as setas do teclado (← →) para navegar entre abas
- Pressione Home para ir à primeira aba
- Pressione End para ir à última aba
- Pressione Escape para fechar o menu mobile

### Personalização
- Edite as cores em `css/styles.css` (variáveis CSS)
- Modifique o conteúdo em `index.html`
- Adicione novas funcionalidades em `js/navigation.js`
